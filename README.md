# HyperPortal

A vibrant dark theme inspired by [LaserWave](https://github.com/Jaredk3nt/laserwave) for Visual Studio Code.

![screenshot](https://github.com/kgwebsites/hyperportal/raw/master/screenshot.png)

**Currently supports:**

- Python
- Javascript
  - React
- Typescript
  - Angular
- Ruby
- Markdown
- Java
- C#
